주식이란?
    회사의 지분을 나눠놓은 조각

    [주식을 보유하고 있다면]
    1. 해당 회사의 의결권을 가짐
    2. 배당금을 받음
    3. 청산시 잔여재산을 받음

    [주식을 사는 방법 - 용어 정리] 주식의 운영 시간 : 9:00 ~ 15:30
    호가
        매도호가 : 사는 것 중에 가장 싼것
        매수호가 : 파는 것 중에 가장 비싼것
    봉차트
        시가 : 장시작 시 가격
        종가 : 장마감 시 가격
        양봉 : 시가보다 종가가 높을 때
        음봉 : 시가보다 종가가 낮을 때
        => 가격이 쌀때 사서 높을때 팔자!

파이썬(투자 전략 구현) -> 작업 스케줄러(증권사 API를 이용하여 자동 매도/매수) -> 알림(Slack 메신저)
투자 전략 구현 : 변동성 돌파 전략을 이용(by 래리 윌리암스 - 전설적인 트레이더)
                변동성 돌파 전략이란?
                    가격의 움직임이 일정 수준 이상을 뛰어넘는 강한 상승세를 돌파 신호로 파악하고
                    상승하는 추세를 따라가며, 짧은 수익을 실현하는 단기 트레이딩 전략(단타)
